Pasos para montar la aplicacion web

1. Ejecutar el script MySQL/Banesco.sql usando MySQL Workbench
2. Ejecutar el script Innova\flask-back_end\flask-back-end\App.py para habilitar el backend.
	- Instalar las dependencias usando pip en caso de no tenerlas
		- flask
		- flask_mysqldb
		- flask_cors
3. Iniciar el front end con npm start adentro de la carpeta react-front_end
